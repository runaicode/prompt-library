# Prompt Library

> Production-tested prompts for AI coding tools, organized by task type.

By [RunAICode.ai](https://runaicode.ai)

## Categories

- **Debugging** — Systematic approaches to finding and fixing bugs
- **Refactoring** — Clean code transformations
- **Code Review** — Prompts for thorough PR review
- **Architecture** — System design and planning prompts
- **Testing** — Test generation and coverage improvement
- **Infrastructure** — DevOps, CI/CD, and deployment prompts
- **Documentation** — Auto-generating meaningful docs

## Usage

Each prompt includes:
- The prompt text (copy-paste ready)
- Which tools it works best with
- Example input/output
- Tips for customization

## Contributing

Share your battle-tested prompts. We verify each one works across multiple AI tools before merging.
